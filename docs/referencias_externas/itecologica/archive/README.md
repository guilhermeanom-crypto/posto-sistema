# Docs - Archive

## Papel desta pasta

Esta pasta guarda documentos historicos da evolucao do Sistema Central V1.

Sao documentos que:

- ja nao orientam sozinhos a operacao atual
- continuam uteis como contexto de origem e como rastro do que foi decidido em cada etapa
- nao devem ser editados como se fossem fonte canonica viva

## Como ler

A leitura operacional canonica esta em `docs/`, na ordem definida em [docs/MAPA_SISTEMA_CENTRAL_V1.md](../MAPA_SISTEMA_CENTRAL_V1.md).

Os arquivos abaixo entram apenas como contexto historico de como o sistema chegou ao estado atual.

## Regra de edicao

- nao editar para refletir mudancas atuais do sistema (esses ajustes vao em docs vivos)
- ao corrigir erro factual ou ortografico, manter a marca de "Status documental: historico" intacta
- ao mover novo doc para ca, atualizar este README e o `MAPA_SISTEMA_CENTRAL_V1.md`
