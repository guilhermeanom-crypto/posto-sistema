# POP-EST-05 — RESULTADO NÃO ESTANQUE, INCONCLUSIVO E TRATAMENTO DE DESVIO

## 1. Se o resultado for NÃO ESTANQUE
- não aprovar;
- registrar o componente específico;
- informar tecnicamente o cliente;
- avaliar necessidade de fluxo de contingência e de integração com passivo/licenciamento;
- manter íntegros todos os registros.

## 2. Se o resultado for INCONCLUSIVO
- justificar tecnicamente o motivo;
- definir se cabe reensaio;
- registrar interferências observadas;
- não emitir conclusão definitiva sem base técnica.

## 3. Desvios proibidos
- alterar dados para favorecer o cliente;
- omitir componente reprovado;
- substituir arquivo não editável por versão editável como evidência principal;
- assinar fora da habilitação/atribuição.
