# KIT EXECUTÁVEL VERSÃO 10/10 — ESTANQUEIDADE SASC (POSTO DE COMBUSTÍVEIS)

Este kit foi montado para estruturação empresarial, operacional e documental de serviços de ensaio e laudo de estanqueidade em Sistema de Armazenamento Subterrâneo de Combustíveis (SASC), com foco em posto de combustíveis e empreendimentos correlatos.

## Escopo do kit
- Implantação do serviço na empresa
- Base regulatória e matriz de requisitos
- Procedimentos operacionais padrão (POPs)
- Checklists de pré-campo, campo e pós-campo
- Modelos de documentos e laudo
- Árvore de decisão para resultados
- Controle de prazos e rastreabilidade
- Painel HTML inicial para navegação offline

## Importante
1. Este kit usa apenas informações obtidas de documentos oficiais e públicos consultados em 2026-03-20.
2. A ABNT NBR 16795:2019 é referência técnica central, mas o texto integral da norma NÃO é reproduzido aqui.
3. Onde a execução depender de detalhe integral da norma ABNT, o kit aponta a necessidade de aquisição/licença da norma pela empresa.
4. Este material NÃO substitui a certificação exigida pelo Inmetro nem a atuação do responsável técnico habilitado.

## Resultado esperado
Ao final do uso deste kit, a empresa terá:
- visão clara do que é obrigatório;
- documentação-base para iniciar a estruturação;
- trilha organizada para auditoria, protocolo e operação;
- modelo de governança para integrar estanqueidade + licenciamento + passivo.
