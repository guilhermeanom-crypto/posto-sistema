# Referências Externas

Esta pasta contém cópias "as-is" de materiais técnicos e de planejamento de projetos e pastas externas consolidadas na Onda 1.

O objetivo é servir de consulta e inspiração para popular dados no banco oficial (Prisma Seeds) ou na inteligência do sistema.

Nenhum código desta pasta é ativamente executado no projeto oficial.

## Estrutura
- `/habilis`: Arquitetura mestre e planejamento.
- `/itecologica`: Documentação da consolidação anterior.
- `/estanqueidade`: Regras, checklists e conhecimento de SASC.
- `/logistica_reversa`: Excel, JSON e mapas mentais do tema.
- `/extintores`: Base técnica para gestão de extintores.
- `/zz_america`: Exemplos reais do Auto Posto Guapó para fins de testes com "mock data" realista.
