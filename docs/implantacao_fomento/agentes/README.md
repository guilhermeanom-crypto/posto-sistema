Este diretorio contem os contratos operacionais dos agentes recomendados para apoiar a consolidacao da base.

Arquivos:

- `01_agente_curador_fontes.md`

- `03_agente_validador_coerencia.md`

- entrada
- saida

- o que nao pode decidir
- formato de entrega
