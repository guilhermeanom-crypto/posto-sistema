# 104. Runbook do Ciclo de Lote

## 1. Abrir lote

1. escolher o lote no backlog
2. nomear dono funcional
3. nomear analista responsavel
4. registrar prazo alvo

## 2. Curadoria

1. reunir fontes
2. consolidar regra aplicavel
3. marcar duvidas
4. aprovar entendimento minimo

## 3. Estruturacao

1. preencher CSV correspondente
2. normalizar chaves
3. revisar datas, status e listas
4. anexar observacoes de excecao

## 4. Validacao

1. revisar duplicidade
2. revisar dependencia
3. revisar obrigatoriedade
4. revisar aplicabilidade
5. decidir `apto` ou `bloqueado`

## 5. Carga

1. executar importacao
2. gerar relatorio de resultado
3. registrar erros por linha
4. repetir ate ficar sem erro bloqueante

## 6. Homologacao

1. conferir amostra no sistema
2. verificar vinculos
3. validar vencimentos
4. validar leitura operacional
5. aprovar lote

## 7. Fechamento

1. marcar lote como concluido
2. registrar licoes aprendidas
3. atualizar backlog seguinte
