# Prompts dos Agentes de IA

Esta pasta abriga os "System Prompts" e regras comportamentais dos agentes do sistema.

Estes arquivos (`.txt` ou `.md`) orientam como o Claude (ou outra LLM) deve ler documentos, extrair metadados, e redigir relatórios técnicos (ex: Agente Coletor, Estruturador, Operacional, Auditor).

Foram migrados do projeto `HABILIS_AI` durante a Onda 1 da consolidação.
