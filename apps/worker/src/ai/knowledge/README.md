# Base de Conhecimento da IA (Knowledge Base)

Esta pasta contém modelos JSON e dados estruturados consumidos pelos agentes de IA durante a análise de documentos.

Eles representam o "o que procurar" e "como classificar" dados específicos de cada domínio (ex: SASC, PGRSS, Áreas Contaminadas).

Foram migrados do projeto `HABILIS_AI` durante a Onda 1 da consolidação.
